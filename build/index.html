This is a part of the final year project done by Tharun Prabhu and Magesh J from the dept. of EEE, SASTRA Deemed to be University. This website is done using simple HTML, CSS (scss) and JavaScript (ESnext). No major libraries are being used. Node.js for the backend and MongoDB for the database. The map integration is accomplished using a well-known library - Leaflet.js . The battery statistics are displayed as a well organized chart with the help of an open source library - Chart.js . The data from the mongoDB database is fed to the front end using mongoose - an open source JS library.
